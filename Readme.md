# Project Name: Konstanz on the Bodensee

A four-page responsive website built and designed to showcase the local area in and around Lake Constance, which is known as the Bodensee. It will show information about upcoming local events as well as locations of nearby major airports for onward travel. The website will focus on clean semantic HTML and custom CSS, with the use of Bootstrap for navigation bar layout efficiency.



## Features

* **HTML/CSS:** Constructed using semantically correct layouts and custom CSS
* **Styling:** Custom color palette, use of unique fonts, effects using custom CSS
* **Responsive Design:** Adaptable layouts that can seamlessly adapt to mobile, tablet and desktop screens.
* **Bootstrap:** Use of Bootstrap for navigation bar styling.



## Site Structure

Website consists of four linked pages:

* `index.html`(Home): A hero section, introduction, video, events list with link to main Events page.
* `events.html`(Events): Showcase upcoming events, with an image, a brief synopsis of each and links directing to external websites.
* `travel.html`(travel): Table of nearby major airports with distance and travel time, a map showing airport locations relative to Konstanz.
* `contact.html`(contact): A contact form with input validation.   


## Technologies Used

* **HTML5:** For semantic page structure.
* **CSS3:** For custom layout styling, colors, and effects.
* **Bootstrap v5.3:** Used strictly for the navbar



## Deployment & Local Setup

### Deployment
Hosted using GitHub Pages. To deploy your own version follow these steps:
1. Navigate to the project repository on GitHub.
2. Click on the **Settings** tab.
3. Scroll down to the **Pages** section on the left sidebar.
4. Under **Build and deployment**, set the source branch to `main` and folder to `/root`.
5. Click **Save**. The live URL will appear at the top of the section shortly.

### Local Cloning
To run this project locally:
1. Clone the repository using `git clone https://github.com`.
2. Open the directory in your preferred code editor.
3. Open `index.html` in any web browser to view the site.
